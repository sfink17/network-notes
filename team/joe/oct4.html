Network Attacks

Internet has become a target of many attacks. Network security deals with defending and protecting networks against attacks.

    Inserting Malware into your host via the Internet

    What is Malware?

    • Software that is intended to damage or disable computers and computer systems.
      • Delete files
      • Install Spyware
      • Intentional harm
    • Viruses
      • Malware that requires some form of user interaction to infect the user's device.
      • Example: Email attachment.
    • Worms
      • Malware that can enter a device without any user interaction
      • Example: Vulnerable network application
    • Botnet -> A network of similarly compromised devices. Used for a DoS attack.
    • Malware is mostly self-replicating

    Attacking servers and Network infrastructure

    Denial-of-service (DoS) attack

    • Vulnerability Attack
      • This involves sending well-crafted messages to a vulnerable application or operating system running on a targeted host
    • Bandwidth Flooding
      • The attacker sends a deludge of packets to the targeted host--so many packets that the target's access link becomes clogged, preventing legitimate packets from reaching the server.
    • Connection Flooding
      • The attacker establishes a large number of half-open or fully opne TCP connections at the target host. The host can become so bogged down with bogus connections that it stops accepting legitimate connections.
    • Distributed DoS

    Sniffing Packets

    Packet Sniffer

    • A passive receiver that records a copy of every packet that passes
    • Sniffed packets can then be analyzed offline for sensitive information.
    • Packet sniffers are passive, so they are very difficult to detect.

    Faking IPs and Identities

    IP Spoofing

    • The ability to inject packets into the Internet with a false source address
    • End-point Authentication
