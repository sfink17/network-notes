DNS: The Internet's Directory Service

  1. Depending on the context, the same entity can be referred to in different ways to gain different kinds of information.
    • Hostnames are the mnemonic addresses for hosts and are the identifier that can be read by humans.
    • IP addresses in IPv4 are fixed length strings of 4 bytes, each a numerical value from 0 to 255, and are the host identifiers that can be read by computers, routers, etc...
  3. DNS (Domain Name System) is part distributed database, part API which gives unique hostnames to web addresses.
  4. DNS is an application level protocol which is used by other application level protocols, one popular program which utilizes this API is BIND (Berkeley Internet Name Domain).
