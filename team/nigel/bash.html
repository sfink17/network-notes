Bash is the default terminal shell on Ubuntu.
Here are some useful commands:
  • cd directory: Moves the terminal to the specified directory.
  • ls [directory]: Lists files in the specified directory (if supplied), or the current directory.
  • pwd: Prints the path of the "working directory"- the directory that the terminal is currently in.
  • cat file: Prints the contents of the file.
  • ping destination: Prints the ms it takes to get a reply from the destination machine.
  • ssh username@host: Opens a terminal shell on the provided host as the provided user.
  • telnet destination port: Opens an unsecure user session to the destination host.
  • mtr destination: Lists a live trace of every stop between your local machine and the destination.
  • whois domain: Prints contact/owner/registration information about the domain.
  • curl --head url port: Prints HTTP header information for the url.
  • nslookup domain: Prints all IPs that host the domain.
  • wget url: Downloads the url into a file.
  • vi file: Edits the file in VIM.
