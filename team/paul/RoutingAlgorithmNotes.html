Notes on Routing Algorithms

  1. Distance-Vector Multicast Routing Protocol

    • This algorithm implements source-based multicast routing with reverse path forwarding and includes pruning techniques.

    • Reverse path forwarding used by this algorithm broadcasts the packets to all surrounding nodes within a specific subset. When the packet is split among nodes, it uses a pruning technique to get rid of the packets heading to nodes that don't need them (they aren't part of the multicast group). Simply put, if a router or node, receives a packet that is directed towards hosts that aren't in it's multicast, then it will let all other routers know by sending prune messages via the upstream route.
