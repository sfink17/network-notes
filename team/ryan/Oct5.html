Lab

  • Linux commands for checking networks
    • netstat: Displays the contents of /proc/net files. The status of ports (open, closed, waiting, etc.).
    • tcpdump: captures packets off the network interface and interprets them (sniffer). Can save packets for later inspection
    • ping: sends ech requests to the specified host on the command line. List responses.
    • hostname: Tells user the host name of the computer
    • traceroute: Shows the route of a packet. Lists the series of host that the packet passes though to the destination.
    • tracepath: same as traceroute, but won't take complicated options
    • findsmb: list info about machines that respond to SMB queries
    • nmap: queries machines as to whether they are up and what ports are open
    • ifconfig: lists all information on all network devices currently up
    • ifconfig eth0 down: takes eth0 down, and won't reciece or send till back up
    • ifup: brings any device back up. ifup eth0 will bring eth0 if it is off
    • ifdown: brings down any device that is currently down
    • ifcfg: configures a particular interface
    • route: displys or modifies the routing table
    • netstat command: displays network connections, routing tables, interfaces and more.
    • ss -s: displays currently established, closed, orphaned and waiting TCP sockets.
    • netstat -s: same as ss -s
    • ss -l: displays all open network ports
    • netstat -tulpn: same as ss -l
    • ss -t -a: displays all TCP sockets
    • netstat -nat:same as ss -t -a
    • ss -u -a: displays all UDP sockets
    • netstat -nau: same as ss -u -a
    • nslookup: queries DNS lookup name
    • telnet: communicate with another hostname
    • finger: view user information
    • ftp: File Transfer Protocol. Standard network protocol to transfer computer files between a client and server
    • rsync: copies files across a network
    • scp: securly copies files and directories between remote hosts wihout starting an FTP session or logging into the remote system, uses SSH to transfer data
    • ssh: program for loggin into a remote machine and for executing commands on a remote machine. Replaces rlogin and rsh
