Questions

  1. Inspect the contents of the first HTTP GET request from your browser to the server. Do you see an "IF-MODIFIED-SINCE" line in the HTTP GET?
    No

  2. Inspect the contents of the server response. Did the server explicitly return the contents of the file? How can you tell?
    Yes. I can see the HTML text in the response. There's also a content type header that lists Text/HTML encoded in UTF-8.

  3. Now inspect the contents of the second HTTP GET request from your browser to the server. Do you see an "IF-MODIFIED-SINCE" line in the HTTP GET? If so, what information follows the "IF-MODIFIED-SINCE" header?
    Yes. Full request URI: http://gaia.cs.umass.edu/wireshark-labs/HTTP-wireshark-file2.html.

  4. What is the HTTP status code and phrase returned from the server in response to this second HTTP GET? Did the server explicitly return the contents of the file? Explain.
    304 Not Modified. No, there is no HTML/Text data in the message, and no content type header.
