4.3

Routers

The transfer of packets from a routers incoming links to the appropriate outbound links at the router. The terms forwarding and switching are often used interchangably.

  • Performs the termination of a physical link at a router.
  • Performs link layer functions needed to interopolate with the link layer at the otherside of the incoming link
  • Lookup function performed at the input port: forwarding table consulted, determines output port for packet to be forwarded to

  • Connects routers input ports to output ports.
  • Completely contained inside router.

  • stores packets recieved from switching fabric
  • transmits packets on outgoing link

  • Executes routing protocols
  • maintains routing tables and link state information
  • computes forwarding table
  • Network management functions

Collective forwarding functions; input ports, output ports, switching fabric

  • Hardware implementation required because software implementation too slow
  • Operates at the nanosecond time scale
  • Execution of routing protocols operates in software
